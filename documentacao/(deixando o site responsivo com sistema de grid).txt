O bootstrap fornece um sistema de grid para que o usuário possa deixar o site todo responsivo. O que significa cada uma das col-* diferentes que o bootstrap define?

1) col-xs-*

2) col-sm-*

3) col-md-*

4) col-lg-*


Questão

Qual o tamanho em px de cada uma das media queries que o bootstrap usa no seu sistema de grid?
Para que é importante sabermos esse valor?

Resposta: XS : < 768px, SM : >= 768px, MD : >= 992px, LG : >= 1200px
É importante saber para quando formos escrever nossas media queries e se quisermos que elas acompanhem os ajustes de tamanho do bootstrap.

col-md-4 para que depois em dispositivos maiores passe a ocupar 3 por linha.

E finalmente col-lg-3 para que em telas grandes ocupem 4 por linha.
